Given a number X. Determine in which of the following intervals the number X belongs to:[0,25], (25,50], (50,75], (75,100]

[0,25] indicates numbers between 0 and 25.0000, including both.

(25,50] indicates numbers greater than 25: (25.00001) up to 50.0000000.

Input
Only one line containing a number X ( - 1000 ≤ X ≤ 1000).

Output
Print the answer to the problem above.

Examples
Input
25.1
Output
Interval (25,50]

Input
25.0
Output
Interval [0,25]
