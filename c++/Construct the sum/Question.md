You are given two integers n and s, and you have to find distinct positive integers, such that each of them is ≤n, and their summation =s. Otherwise, state that this is impossible.

Input
The first line contains a number T (1≤T≤100) – number of test cases.

Each of the next T lines contains two space-separated integers n,s (1≤n≤10^5,1≤s≤101^8).

Output
For each test case, if there is no possible answer, print −1 on a single line.
Otherwise, print the set of numbers that satisfy the above condition on a single line. 
You can print the numbers in any order. If there are multiple answers, you can print any of them.

Example
Input
4
5 3
7 10
6 4
2 10
Output
2 1 
4 3 2 1 
3 1 
-1
