You are given an array a of n integers, and an integer k

You can choose at most k elements and get their summation.

What is the maximum summation you can get?

Input
The first line contains two integers n and k(1≤k≤n≤10^3) the number of elements in the array a and the maximum elements you can choose respectively.

The second line contains n integers ai(−109≤ai≤10^9) the elements of the array a.

Output
Output the maximum summation you can get.

Examples
Input
2 2
1 2
Output
3

Input
2 1
1 2
Output
2

Input
3 3
1 2 3
Output
6
